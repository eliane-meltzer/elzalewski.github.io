Each year, Compassion International releases a President's Report that goes over all the major happenings in the organization. This year, the Communications Department wanted to shake things up a bit and release an web version to go along with it! My role as an intern was to take the print version and translate it into a website so that viewers could view the information easier.

Design Studies

The color scheme and typography mimicked the report.

Print Inspiration

The entire report is 33 pages long! One challenge was figuring out how to create visual continuity between the print and web versions while also taking advantage of web capabilities that would enhance user experience.
