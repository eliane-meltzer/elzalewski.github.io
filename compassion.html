Compassion International

Summer 2018 | Interactive Report | URL: https://presidentsreport.compassion.com | *email for password

Each year, Compassion International releases a President's Report covers all the major developments in the organization. This year, the Communications Department wanted to shake things up a bit and release an web version that would enhance the typically static report. My role as an intern was to take the print version and translate it into a website that would be more readable and interactive for our viewers.
